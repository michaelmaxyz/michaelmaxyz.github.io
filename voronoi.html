Delaunay triangulations using Bowyer-Watson

A delaunay triangulation is a triangulation such that every circumcircle of every triangle have empty interiors. A brute force approach is slow, running at O(n^4) time (checking EVERY possible circumcircle using EVERY triplet of points and then checking if EVERY other point is outside of said circumcircle), but there exists a O(n^2) approach and if optimized, a O(nlogn) algorithm using Bowyer-Watson. Bowyer-Watson is an incremental algorithm that begins with a "supertriangle" that contains every point. We then add points one by one and we delete every new triangle created that contains a new point. We then connect edges from the new point to every other point until every point has been added. We then remove the super triangle and its corresponding triangles.

A voronoi diagram is simply a diagram where there exists a finite set of points and for each point there exists a Voronoi cell that consists of all locations closer to it than any other point. A voronoi diagram can be easily derived from a delaunay triangulation. For each triangle, the circumcenter becomes the vertices of the voronoi diagram, and for every adjacent set of triangles, simply connect the circumcenters of the triangles and you get a voronoi diagram.
