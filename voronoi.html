Dynamic Voronoi Diagram

A voronoi diagram is simply a diagram where there exists a finite set of points and for each point there exists a Voronoi cell that consists of all locations closer to it than any other point. Each voronoi cell is colored a random color and the program allows you to remove points using the "A" key and add points using the "D" key. Programmed in java. Source code on Github.
