1. Arithmetic

1.1 Commutative Property

$$a + b = b + a$$ $$a * b = b * a$$

1.2 Subtraction is Addition, Division is Multiplication

$$a - b = a + (-b)$$ $$\frac{a}{b} = a * (\frac{1}{b})$$

1.3 Identity Property

$$a + 0 = a$$ $$a * 1 = a$$

1.4 Associative Property

$$(a * b) * c = a * (b * c) = (a * c) * b$$

1.5 Distributive Property

$$a(b + c) = ab + ac$$

1.6 Multiplying Fractions

$$a(\frac{b}{c}) = \frac{ab}{c}$$ $$\frac{a}{b}*\frac{c}{d} = \frac{ac}{bd}$$

1.7 Dividing Fractions

$$\frac{\frac{a}{b}}{c} = \frac{a}{bc}$$ $$\frac{a}{\frac{b}{c}} = \frac{ac}{b}$$ $$\frac{\frac{a}{b}}{\frac{c}{d}} = \frac{ad}{bc}$$

1.8 Adding and Subtracting Fractions

$$\frac{a}{b} + \frac{c}{d} = \frac{(ad + bc)}{bd}$$ $$\frac{a}{b} - \frac{c}{d} = \frac{(ad - bc)}{bd}$$

1.9 Combining common denominators

$$\frac{a + b}{c} = \frac{a}{c} + \frac{b}{c}$$

1.10 Getting rid of denominators

$$\frac{ac + bc}{c} = a + b$$

1.11 Dividing by 0

\(\frac{a}{0} = undefined\)
In certain disciplines, something divided by 0 would actually become \(\infty\) because 0 would be a VERY small number, and dividing something by a VERY small number creates a VERY BIG number (infinity).

2. Exponents

2.1 Definition of an Exponent

$$a ^ 2 = a * a$$ $$a ^ 3 = a * a * a$$ .. and so on.

2.2 Multiplying exponents of same base

$$a^m*a^n = a^{m+n}$$

2.3 An exponent to an exponent

$$a^{b^c} = a^{bc}$$

2.4 A base with more than 1 variable

$$(a*b)^c = a^c*b^c$$

2.5 Negative Exponents

$$a^{-b} = \frac{1}{a^b}$$

2.6 Exponents with Fractions

$$(\frac{a}{b})^c = \frac{a^c}{b^c}$$

2.7 Fractions with Same Base

$$\frac{a^b}{a^c} = a^{b-c}$$

2.8 Power of Zero

$$a^0 = 1$$

3. Logarithms

3.1 Definition of a Logarithm

\(a^b = c\) is equivalent to \( \log_a{c} = b \). We would say this in English as "log base a of c equals to b"
